# Wrong Parts Shipped — Response Pattern

Source: Discovery call with Dan, 2026-04-06

## Customer Question Pattern
"I got the wrong parts — ordered [SKU A], received [SKU B]"
"The car came with the wrong tires"
"I ordered X but got Y"

## Response Pattern
- Apologize for the error
- Send the correct parts immediately
- Provide a return label for the wrong item
- OR: send a replacement car and have customer return the incorrect one (for wrong tires on cars)

## Example — SKU 432 Phantoms vs SKU 444 Black Tires
Wrong tires were on a car. Dan's approach: either send correct tires or send replacement car and have customer return the incorrect one.
