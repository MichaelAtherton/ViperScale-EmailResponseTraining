# Timing System Setup — Response Pattern

Source: Discovery call with Dan, 2026-04-06

## Customer Question Pattern
"I received my tracking system. When it comes time to set up, can I ask questions?"
"How do I set up the timing software?"
"I'm having trouble with the timing system software"

## Response Pattern
- Provide link to software setup video (on the timing system product listing)
- Provide link to software help/tech support section
- Note: timing system is **universal** — works on any brand track (Viper, AFX, etc.)
- Encourage customer to review materials before the product arrives

Dan's approach: "if you have questions, here's a video. Here's also the tech support paperwork"

## Windows S-Mode Issues
- Some customers have Windows in S-mode which prevents installing the timing software
- Viper does NOT provide Microsoft support
- Dan: "I don't want to provide Microsoft support for him"
- Provide a link to Microsoft's instructions for switching out of S-mode into home mode
- Don't try to troubleshoot the OS issue
