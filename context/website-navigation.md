# Website Navigation — viperscaleracing.com

Source: Site crawl, 2026-04-07 (208 product categories from sitemap)

## How to Find Products

The primary navigation is **"Shop by Car Type"** — select a chassis type, then browse subcategories.

Path: **Shop by Car Type → [Car Type] → [Category]**

Example: Shop by Car Type → Tomy AFX → Magnatraction → Tires

**Key principle:** "Anything that's in here will fit" — everything in a car type's section is compatible with that car type.

## Search Behavior
- The site search can be limited — customers often can't find items by searching specific part names
- Better to browse by car type category than to search by specific part name
- Dan's exact words: "The search engine doesn't seem to find these"
